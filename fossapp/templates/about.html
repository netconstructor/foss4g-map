Contribute

All of the maps you see on this site are created from OpenStreetMap data. OpenStreetMap is a free wiki map of the world. This means that you can contribute your local knowledge to a map that can used by anyone. There is a Denver OpenStreetMap meetup group that you should join if you are interested in OpenStreetMap.

Open Source + Open Data

This application would not have been possible if it weren't for a number of open source and open data projects. Do yourself a favor and check out a few of the greate projects below.

Leaflet

Leaflet is a JavaScript web mapping library from CloudMade that creates the "slippy" map interface. This map can be easily customized by adding different base maps, overlays, markers, lines, etc. Leaflet is similar to products like OpenLayers and the Google Maps API, but is different as it is both light weight and open source making this a good solution for mobile applications or disconnected use.

  • Leaflet Website
  • Leaflet Source Code

TileMill

TileMill is a browser-based web map design studio from the folks over at MapBox. TileMill lets you create beautiful maps using a CSS-like syntax, which is much more favorable than using SLD. Maps can be exported in an open mbtiles format, which is just a SQLite database. Map tiles can be served from this file from a few different software packages including MapBox's TileStream and TileStache.

MapFish

MapFish is an open source framework for creating web mapping applications. It is built on the Pylons web framework and integrates tightly with a number of other open source projects (SQLAlchemy, GeoAlchemy, Shapely). MapFish works by creating web services that can Create, Read, Update and Delete (CRUD) geographic features via HTTP POST, GET, PUT and DELETEs.
